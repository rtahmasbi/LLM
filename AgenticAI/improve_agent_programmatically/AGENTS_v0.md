# AGENTS.md

## Setup

Install the project dependencies:

```bash
pip install -e .
```

## Running Tests

Run the test suite with pytest:

```bash
pytest
```

## Linting

Use ruff to lint the code:

```bash
ruff check .
```

## Code Style

Follow PEP 8. Use type hints where possible. Write docstrings for public functions.

## Pull Requests

- Write clear commit messages
- Add tests for new features
- Make sure CI passes before merging
